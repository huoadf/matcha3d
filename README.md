# Matcha3D — 3D World Shapes ESP & Dynamic Web Mesh Engine

A high-performance 3D Drawing shape engine built for Roblox LuaVM / Matcha executor. Features real-time parametric 3D rendering, Target-Lock ESP tracking, physics, trails, outlines, and a dynamic GitHub Pages Web Shape loader.

## 🚀 Features

- **18+ Parametric 3D Shapes**: Cone, Cube, Sphere, Diamond, Torus, Star, Crown, Pyramid, Cylinder, Capsule, Helix, Icosahedron, Tetrahedron, Prism, Arrow, Disc, Cross / Star of David, **Möbius Strip**, **Torus Knot (3,2 Trefoil)**, **Hyperboloid / Hourglass**, and **Custom Web Shapes**.
- **🎯 Target-Lock / ESP Tracking**: Attach 3D shapes to track or orbit targeted players or closest enemies.
- **🌐 Dynamic Web & JSON Loader**: Load custom 3D mesh definitions dynamically over HTTP from GitHub Pages or raw JSON code.
- **✨ Visual FX**: Wireframe mode, Rainbow colors, Outline glow, Dual-color Ping-Pong cycle, Speed scaling, Height heatmap, Chromatic aberration split, Pulsing, Shake/jitter, Idle drift walk, and Echo sonar pulses.
- **🎛️ INS-UI Interface**: Complete in-game UI menu with real-time sliders and keybinds.

---

## 📜 How to Execute

Run the script in your LuaVM executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/huoadf/matcha3d/main/shapes.lua"))()
```
